RAYTWIT

A one of a kind twitter scraper that bypasses all of twitters APIS and manages to grab all data metrics

RayTwit is a tool made by Raiyaan Bawany, offering groundbreaking capabilities in Twitter data scraping. By bypassing API limitations, RayTwit retrieves a wealth of metrics, including tweet screenshots, providing unmatched insights into Twitter's realm. Raiyaan Bawany's creation revolutionizes data mining, offering a new frontier for exploration and analysis on the platform.
